# QR Code Generator
Generate QR Codes easily.

## Setup
### Requirements
- Node.js (v20)
- Bun (optional. 1.1.25 or above)
- Git

```sh
git clone https://github.com/martinval11/qr-code-generator
cd qr-code-generator
bun install # or npm install
```
Then, open [localhost:5173](http://localhost:5173)
